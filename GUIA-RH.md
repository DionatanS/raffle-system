# 🎲 Sistema de Sorteio Interno — Guia Rápido para o RH

Olá! Segue abaixo o passo a passo para utilizar o sistema de sorteio. O sistema está em **fase de teste**, então qualquer dúvida ou problema pode reportar diretamente.

---

## 🔐 Acesso ao Painel

**Endereço:** `http://localhost:3000` *(ou o link que for informado pela TI)*

| Campo   | Valor             |
|---------|-------------------|
| Usuário | `rh.admin`        |
| Senha   | `Rh@Sorteio2026!` |

---

## ▶️ Como Criar um Sorteio

1. Acesse o sistema e faça login com as credenciais acima
2. No painel, informe a **quantidade de números** que o sorteio terá
3. Clique em **"Criar Sorteio"**
4. Um **link único** será gerado — copie e envie para os participantes

---

## 👥 O que o Participante Faz

O participante acessa o link recebido e:

1. Visualiza a grade de números disponíveis
2. Clica no número que deseja escolher
3. Informa o **nome** no campo que aparece
4. Confirma — o número fica reservado para ele

> Cada participante só pode escolher **um número por sorteio**. Números já reservados ficam bloqueados para todos automaticamente.

---

## 🎯 Realizar o Sorteio

- O botão **"Sortear Ganhador"** é liberado automaticamente quando **90% ou mais** dos números estiverem reservados
- Clique no botão para sortear
- O ganhador é exibido **para todos** ao mesmo tempo (RH e participantes)
- O resultado é salvo no **Histórico de Sorteios**

---

## 🔄 Após o Sorteio

- Para realizar um novo sorteio, clique em **"🔄 Iniciar Novo Sorteio"**
- O sorteio anterior é encerrado e você pode configurar um novo

---

## ℹ️ Observações

- O painel do RH atualiza os participantes **automaticamente a cada 2 segundos**
- O link de cada sorteio é **único** — links de sorteios anteriores ficam inválidos
- Em caso de necessidade, é possível **cancelar um sorteio em andamento** pelo painel
- O sistema está em fase de testes — eventuais instabilidades estão sendo monitoradas
